# Kemal Selam Friends — Cleaning Website

Premium marketing site for **Kemal Selam Friends Cleaning Services** (Deliver Agency): React, Vite, Tailwind CSS, Framer Motion.

**Tagline:** Cleanliness with Friendship and Care

## Setup

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18, React Router, Vite
- Tailwind CSS, Framer Motion, Lucide React
- WhatsApp links via `wa.me` (see `src/config/whatsapp.js`)
